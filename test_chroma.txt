There is no separate ChromaDB server you log into or query directly like you would with PostgreSQL using psql.
Think of it like SQLite — you never "connect to SQLite" as a separate running service. You just import it in Python and it reads/writes files directly. ChromaDB works the same way.
python# PostgreSQL — needs a running server, connection string, credentials
import psycopg2
conn = psycopg2.connect("host=localhost dbname=mydb user=postgres")

# SQLite — just a file, no server needed
import sqlite3
conn = sqlite3.connect("./mydb.sqlite")

# ChromaDB — same idea as SQLite, just a local file
import chromadb
client = chromadb.PersistentClient(path="./data/chroma_db")
